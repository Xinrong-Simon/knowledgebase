# 2025年芬兰初创公司重大融资报告

> 2025年，芬兰初创公司融资总额创历史新高，达 **19亿欧元**，较2024年增长约65%。以下为2025年融资超100万欧元的代表性公司详情。

## 整体数据总览

| 指标 | 数值 |
|------|------|
| 2025年融资总额 | **约19亿欧元**（历史最高） |
| 较2024年增幅 | **+65%** |
| 最大单笔融资 | Oura（€777M，E轮） |
| 最热门赛道 | 量子计算、航天科技、AI基础设施、食品科技、健康科技 |
| 外资占比 | 约75% |
| 芬兰最大VC基金 | Lifeline Ventures（4.25亿欧元，史上最大） |

---

## 公司详情

### 1. Oura Health

| 项目 | 详情 |
|------|------|
| **成立时间** | 2013年 |
| **总部** | 奥卢（Oulu），芬兰 |
| **融资轮次** | E轮 |
| **融资金额** | **约€777M（~9亿美元）** |
| **融资时间** | 2025年10月 |
| **领投方** | Fidelity Management & Research Company |
| **跟投方** | ICONIQ、Whale Rock、Atreides |
| **融资后估值** | **110亿美元**（欧洲首家健康追踪十角兽） |

**创始人**
- Petteri Lahtela — 联合创始人兼前CEO
- Virpi Tuomivaara — 联合创始人
- Kari Kivelä — 联合创始人
- Markku Koskela — 联合创始人

**公司业务**

Oura 开发了一款智能指环（Oura Ring），可全天候监测用户的睡眠质量、心率、体温、活动量及压力水平，提供个性化健康洞察。公司已累计销售超550万枚设备，2024年营收达5亿美元，预计2025年突破10亿美元。

|------|------|

| **总部** | 埃斯波（Espoo），芬兰 |

- Kuan Yen Tan — 联合创始人兼CTO

**公司业务**

IQM 是欧洲最领先的量子计算公司，提供本地部署全栈量子计算机和云端平台。本轮为欧洲及美国以外最大的量子计算B轮，将用于推进量子纠错技术，目标从千量子比特扩展至百万量子比特系统。本轮完成后 IQM 跻身独角兽行列。
